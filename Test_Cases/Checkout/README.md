# Check - Test Case Prehľad

Tento priečinok obsahuje všetky pozitívne a negatívne testovacie prípady ktoré som navrhol a použil pri testovaní webu SauceDemo.com. Cieľom bolo zisťiť pomocou testov či fungujú základné princípy checkout podla plánovaných scénárov.

---

## Štruktúra

  – [Checkout](./Checkout.md)  
  – valídne zobrazovanie prechodu na Checkout

## Zameranie

Test Case sú zamerané na :

- **Validácia polí** - štruktúra, povinné polia
- **Autentifikačná logika** - platné a neplatné pokusi o odoslanie formulára
