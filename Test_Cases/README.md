## Prehľad sady webových testov

Táto sekcia obsahuje štruktúrované testovacie prípady pre webovú aplikáciu. Každá sekcia je usporiadaná do vlastného priečinka a obsahuje pozitívne a negatívne scenáre.

---

## Stratégia vykonávania
Všetky testovacie prípady sú napísané pre manuálne testovanie, dodržiavajú konzistentný formát a sú jednoducho opakovaťelné scenáre

- Predpoklady
- Jasný postup a jednotlivé kroky
- Očakávané výsledky  
- Reálne výsledky
